# I2POC Contract Versions

This repository contains two versions of the I2POC Contract application.

## Version 1
- Original version of the I2POC Contract application.
- Contains backend (idea_be) and frontend (idea_fe) with basic functionality.

## Version 2
- Enhanced version with additional features:
  - Contract generation service
  - Document processing service
  - Upload tab fixes
  - Improved logging and testing

## Structure
- `version1/` - Original I2POC Contract
- `version2/` - Enhanced I2POC Contract with new functionality

Each version includes:
- Backend (Python/Flask) in `idea_be/`
- Frontend (React/Vite) in `idea_fe/`
- Configuration files, static assets, and documentation

## Getting Started
Refer to individual version README files for setup instructions.
